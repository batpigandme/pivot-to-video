tidyr::pivot_longer()

Since I only want to elongate the movies, and all of those columns are logical, Iâ€™ll first select which columns I want by using select_if() and is.logical() as the predicate function.

  colnames()

movie_long <- movie_results %>%
  pivot_longer(
    cols = one_of(logicols),
    names_to = "movie",
